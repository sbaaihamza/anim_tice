# anim_tice

## Installation

To run the animations, you need to install the `anim_tice` package in editable mode:

```bash
pip install -e .
```

## Running an Animation

To run a specific lesson, use the `manim` command:

```bash
manim -pqh M3_G05_Ruler_Construct_Segments_And_Lines.py M3_G05_Ruler_Construct_Segments_And_Lines
```
